# GameFeed :

Réseau social centré sur le jeu vidéo — Projet de fin de formation DWWM.

## Stack technique :

- **Front-end** : React, Vite, React Router, Axios
- **Back-end** : Node.js, Express, MySQL
- **Authentification** : JWT

## Structure du projet :

- `client/` — Interface utilisateur React
- `server/` — API REST Express

## Lancement :

Se référer aux README dans les dossiers `client/` et `server/`.
